# 时间轴歌词制作工具

一个无需上传音频文件的在线时间轴歌词制作工具，帮助您轻松为歌词添加时间轴标记，生成标准 LRC 格式歌词文件。

## ✨ 功能特点

- 🎵 **无需音频文件**：直接使用外部音乐播放器同步制作
- ⏱️ **高精度计时器**：支持毫秒级时间标记
- ⌨️ **快捷键支持**：空格键快速记录时间
- 🎯 **智能过滤**：自动过滤 Suno 提示词（如 `[Chorus]`、`（编曲提示）` 等）
- 📋 **一键复制**：支持复制原歌词和带时间轴歌词
- 💾 **LRC 导出**：生成标准 LRC 格式文件，兼容各种播放器

## 🚀 快速开始

### 在线使用

直接在浏览器中打开 `index.html` 即可使用，无需安装任何依赖。

### 本地开发

```bash
# 克隆项目
git clone <repository-url>

# 进入项目目录
cd 制作歌词

# 使用浏览器打开 index.html
# 或使用本地服务器（推荐）
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 📖 使用方法

1. **输入歌词**：在文本框中粘贴您的歌词
2. **选择过滤选项**（可选）：勾选需要过滤的提示词类型
3. **点击"开始制作"**：预览歌词内容
4. **开始制作时间轴**：点击按钮或歌词预览区域打开制作弹窗
5. **播放音乐**：在外部播放器中播放音乐
6. **记录时间**：听到每句歌词开始时，按空格键或点击按钮记录时间
7. **修改调整**：点击已记录的歌词行可以重新修改时间（自动回退2秒）
8. **完成制作**：点击"完成制作"查看结果
9. **导出/复制**：复制歌词或下载 LRC 文件

## ⌨️ 快捷键

| 按键 | 功能 |
|------|------|
| `空格` | 开始/暂停计时，或记录当前时间 |
